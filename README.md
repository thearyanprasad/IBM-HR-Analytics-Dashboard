# 📊 IBM HR Analytics Dashboard

## Overview

This project is an interactive HR Analytics Dashboard developed in Power BI using the IBM HR Employee Attrition dataset.

The dashboard provides insights into workforce demographics, employee attrition trends, performance metrics, compensation analysis, and employee satisfaction.

---

## Dashboard Pages

### 1️⃣ HR Overview Dashboard

Provides a high-level overview of workforce metrics including:

- Total Employees
- Attrition Rate
- Average Age
- Average Income
- Department Distribution
- Gender Distribution
- Education Field Analysis
- Age Group Analysis

---

### 2️⃣ Attrition Analysis Dashboard

Focused on identifying factors contributing to employee turnover.

Key analyses include:

- Attrition by Department
- Attrition by Job Role
- Attrition by Overtime
- Attrition by Age Group
- Employee Retention Metrics

#### Key Findings

- Attrition Rate: 16.12%
- Overtime employees account for 53.6% of attrition
- Laboratory Technicians experience the highest turnover
- Employees aged 20–30 show the highest attrition

---

### 3️⃣ Performance & Compensation Dashboard

Analyzes employee performance, salary distribution, and satisfaction metrics.

Includes:

- Average Salary by Department
- Average Salary by Job Role
- Job Satisfaction Distribution
- Environment Satisfaction Distribution
- Salary Hike Analysis
- Performance Rating Analysis

#### Key Findings

- Managers receive the highest average salaries
- Most employees report satisfaction levels of 3–4
- Environment satisfaction remains generally positive
- Salary hikes remain within expected HR benchmarks
